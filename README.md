## 📖 About

Construction Knowledge Atlas visualizes **13,000+ academic papers** from major construction technology journals:

- **Technology Ontology** — Radial hierarchy of construction tech keywords (OpenAlex lv0-5)
- **Country Analysis** — World map of research output + institution keyword ontology
- **Institution Network** — Co-authorship network (Force 2D / 3D / Geographic map)
- **Impact Papers** — Top papers ranked by citations + FWCI + recency
- **Tech Evolution** — Monthly emergence tracking + Gantt timeline
- **Knowledge Graph Summary** — Theme-level overview with drill-down

## 🔬 Data Sources

9 journals via [OpenAlex API](https://openalex.org/): Automation in Construction, Advanced Engineering Informatics, Construction and Building Materials, Journal of Building Engineering, Buildings, Frontiers in Built Environment, KSCE Journal of Civil Engineering, ITcon, JCEM.

## 🛠️ Stack

- **Frontend**: Vanilla JS + D3.js (single HTML file, no build step)
- **Data**: Python + OpenAlex API + OpenAlex concept hierarchy
- **Hosting**: Cloudflare Pages (auto-deploy on push)
- **Analytics**: Google Analytics 4
